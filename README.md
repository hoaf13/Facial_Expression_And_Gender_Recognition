## Installation 
- Clone this project: 
 `$ git clone https://github.com/hoaf13/Facial_Expression_And_Gender_Recognition.git `
- Install **virtualenv** and run command:
`$ pip install -r requirements.txt`
- Run file main: `$ python3 main.py`
